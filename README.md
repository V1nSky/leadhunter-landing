## What This Project Shows

- Product thinking around Telegram-based lead generation.
- Landing page presentation for a Telegram Mini App concept.
- Dashboard-style UI with product screens and interaction states.
- Multi-account parsing/monitoring concept without claiming public commercial launch.
- Automation and outreach workflow positioning.
- Clear product messaging for recruiters, clients, and portfolio visitors.

## Product Idea

LeadHunter is presented as a product/case for finding, processing, segmenting, and exporting leads from open Telegram sources. The landing explains the value of combining Telegram accounts, source monitoring, lead lists, niches, exports, and operational dashboards into one product flow.

## Local Run

This is a static GitHub Pages landing without a build system.

```bash
open index.html
```

Alternative with a local static server:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```
